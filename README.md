# BeAnimal Web

브라우저에서 웹캠을 열고, *사람 전체를 동물 실루엣으로 동물화*하는 웹앱입니다.
이 버전은 단순 얼굴 스티커가 아니라, ML 세그멘테이션으로 피사체의 몸 전체를 재구성하는 쪽으로 바꿨습니다.

## 바로 실행

### 1) 로컬에서 보기
Vercel용 루트는 `web` 입니다.
개발/확인은 `web` 안에서 실행하시면 됩니다.

```bash
cd web
npm install
npm run dev
```

또는 정적 서버로 보면:

```bash
cd web/dist
python3 -m http.server 8000
```

그다음 브라우저에서 아래 주소를 여세요.

- http://127.0.0.1:8000/

## 배포된 웹앱
- Vercel 프로덕션: https://be-animal.vercel.app

Vercel에서 가져갈 때는 *Root Directory* 를 `web` 으로 잡으세요.
저장소 루트에는 Python 데스크톱용 파일이 남아 있어서, 루트 그대로 import하면 프로젝트 타입이 꼬일 수 있습니다.

## 지원 동물
- 카피바라
- 고양이
- 햄스터
- 백곰
- 진돗개
- 호랑이

## 기능
- 브라우저 웹캠 미리보기
- ML 기반 사람 세그멘테이션
- 몸통/머리/팔다리를 동물 실루엣으로 재구성
- 숫자키 `1~6` 또는 버튼으로 동물 변경
- `s` 키 또는 버튼으로 사진 저장
- 카메라가 없어도 미리보기 모드로 효과 확인 가능

## 사용 방법
1. 브라우저에서 앱을 엽니다.
2. `카메라 시작`을 누릅니다.
3. 카메라 권한을 허용합니다.
4. 사람을 비추면 실시간으로 동물화됩니다.
5. 동물을 바꾸려면 위 버튼이나 숫자키 `1~6`을 누릅니다.

## 조작
- `1~6`: 동물 변경
- `c`: 카메라 시작/중지
- `s`: 사진 저장

## 검증용 힌트
브라우저 콘솔에서 `window.__beanimal.renderPreview()` 를 호출하면 미리보기 동물화 상태를 다시 그릴 수 있습니다.

## 참고
- 실제 카메라 환경에서는 ML 세그멘테이션이 실시간으로 돌고, 미리보기는 카메라 없이도 동작합니다.
- 기존 Python/OpenCV 데스크톱 버전 코드는 레거시로 남겨 두었습니다.
- 지금의 기본 웹 진입점은 `web/index.html` 입니다.
