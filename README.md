# BeAnimal Web

브라우저에서 웹캠을 열고, 얼굴 위에 **동물 필터**를 씌우는 웹앱입니다.
Python 없이도 바로 열 수 있게 만들었습니다.

## 바로 실행

### 1) 로컬에서 보기
정적 서버를 하나 띄우고 브라우저로 여시면 됩니다.

```bash
python3 -m http.server 8000
```

그다음 브라우저에서 아래 주소를 여세요.

- http://127.0.0.1:8000/

### 2) 배포된 웹앱
GitHub Pages 또는 다른 정적 호스팅에 올리면 바로 접속형 웹앱이 됩니다.

## 지원 동물
- 카피바라
- 고양이
- 햄스터
- 백곰
- 진돗개
- 호랑이

## 기능
- 브라우저 웹캠 미리보기
- 얼굴 감지 후 동물 필터 오버레이
- 숫자키 `1~6` 또는 버튼으로 동물 변경
- `s` 키 또는 버튼으로 사진 저장
- GitHub Pages 같은 정적 호스팅에 바로 배포 가능

## 사용 방법
1. 위의 로컬 서버를 띄웁니다.
2. 브라우저에서 BeAnimal Web을 엽니다.
3. `카메라 시작`을 누릅니다.
4. 카메라 권한을 허용합니다.
5. 동물을 바꾸려면 위 버튼이나 숫자키 `1~6`을 누릅니다.

## 조작
- `1~6`: 동물 변경
- `c`: 카메라 시작/중지
- `s`: 사진 저장

## 검증용 힌트
브라우저 콘솔에서 `window.__beanimal.previewFace()` 를 호출하면 카메라 없이도 오버레이를 시험할 수 있습니다.

## 참고

- 기존 Python/OpenCV 데스크톱 버전 코드는 레거시로 남겨 두었습니다.
- 지금의 기본 진입점은 `index.html` 웹앱입니다.
